# Task 07 - Console Email Delivery

## Goal

Support local email authentication by printing codes to the console.

## Scope

- Add console email delivery provider.
- Support only `EMAIL`.
- Print destination and code.
- Make it selectable by configuration.

## Acceptance Criteria

- Local provider sends by writing to console output.
- Provider is suitable for local development and tests.
- Provider is not intended for production use.
